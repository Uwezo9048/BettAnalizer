# 🤖 AI Betting Assistant

An intelligent betting assistant that fetches live odds from multiple betting sites (SportyBet, Bet9ja, 22bet, Paripesa, Nairabet, Betking) and uses AI to detect value bets, calculate expected value, and suggest optimal stakes using the Kelly Criterion.

## ✨ Features

- **Live Odds Fetching** - Real-time odds from 6+ betting sites via OddsAfrica-API
- **AI Value Detection** - Calculates expected value (EV) for every market
- **Multiple Markets** - 1X2, Over/Under, GG/NG, Double Chance, Handicap
- **Kelly Criterion Staking** - Optimal stake calculation for bankroll management
- **Manual Entry** - Input odds from any site manually for analysis
- **Searchable Site Dropdown** - Easy selection of supported betting sites
- **Real-time Analysis** - One-click analysis of all matches
